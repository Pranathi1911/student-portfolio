Railway Reservation System

The Railway Reservation System in C is a console-based application enabling users to securely book, view, and cancel train tickets. It features user authentication for admin and user roles, allowing access to different functionalities. Users can search for trains, book tickets by selecting a train and entering passenger details, and receive a unique booking ID for each reservation. They can also view all booked tickets and cancel reservations using the booking ID. Admin functions include adding, modifying, and deleting train information, as well as viewing all reservations. The system utilizes basic data structures like arrays and structures for managing train and booking data.

Student Record Management System

The Student Record Management System in C++ is a console-based application designed to manage student information efficiently. It allows users to add, view, update, and delete student records. Each record includes details such as student ID, name, age, grade, and contact information. The system utilizes file handling to store and retrieve data, ensuring persistence across sessions. It also features a user-friendly menu-driven interface, making it easy to navigate and perform various operations. This system is ideal for schools and educational institutions to maintain organized and accessible student records.
